XML2Contacts

Version 0.4 - with JPG photo import

N.B:

This webapp restores your contacts using previous backups generated with Contacts2XML webapp. It has been tested with hundreds of contacts, but is still under development.

Don't forget to unplug any USB wire before restoring a backup or you won't be able to read from the sd-card !

Please note that no comparisons are made between new contacts and existing ones, which means you can add same contacts several times !
