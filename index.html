N.B.:

  • No comparison is made between existing contacts and new ones. To avoid multiple records, you can clear the current database with the following button before restoring your datas, but remember it really clears ALL of your existing contacts definitely !

  • Backups should be placed in the directory sdcard/contacts2xml/YYYYMMDDHHIISS.
  • External photos made with a version of Contacts2XML<0.5 should be rename from "XXX.jpg" to "XXX.1.jpg".
  • You can use Cockatoo to create and manage your contacts outside of Firefox OS. Since version 0.5, Cockatoo read/write contact photos using data-uri format.
  • Multiple photos are already supported by Firefox OS API, but current versions only show a single photo per contact.
